# Calendar New Tab

Chrome extension with an Ethiopian calendar grid, Geez analog clock, holidays, todos, and countdowns.

## Load unpacked

```bash
npm install
npm run build
```

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. **Load unpacked** → select the `dist` folder

Opening a new tab replaces Chrome’s default page with this UI (full page, no Chrome NTP footer bar).

## Language modes

- **Combo** (default): calendar dates, months, and holidays in Amharic; UI chrome in English
- **English** / **Amharic**: full UI in that language

## Privacy

Todos, countdowns, and settings stay on your device. Weather uses Open-Meteo only.
